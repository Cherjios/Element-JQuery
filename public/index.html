How to create an element on html with jQuery?

First of all, let me describe what exactly is jQuery.

According to the website https://jquery.com is a JavaScript library that makes it easier to manipulate HTML documents, use animation, and a much simple way to use Ajax to get data from APIs, so in other words simplify the way to write on JavaScript.

To create an element on an HTML document with jQuery we need the use the dollar symbol to refer to the HTML element we want to create on our HTML file.

For example, if you want to create a simple header “Hello word” on your HTML document, you will use the

tag as fallow.

<h1>"Hello World"</h1>

This instruction will create a “Hello world” on your browser and every time you reload or refresh your website this header is going to show up. However this header is going to be there all the time and sometimes we do not need that, here is when jQuery or JavaScript comes in handy.

In order to create an element for example:

“Hello world!”

on JavaScript we need to fallow the next instructions:

var h1Element = $("<h1>").text("Hello World");

Now that we have created this element, we need to append it to another element in the HTML document to be able to show it on the browser. To append it we can use the JavaScript method append() or prepend().

append(). Will add the element at the end.

prepend(). Will add the element at the beginning.

For example: Let’s say we have a <div id="headerText">...</div> in our HTML file and we want to add the “Hello world” to our document.

We need to make reference to the element, class, or id where we want to add the new element by calling the dollar symbol first, then the address element where we are putting the new element followed by the dot (.) and the method that we want to use( append if we want the element at the end or prepend if we want the element at the beginning).

$(“#headerText”).append(h1Element);

$(“#headerText”).prepend(h1Element);

o, our HTML document being opened on the browser will show us the next code:

“Hello, World” and if we inspect the element in our console we will see as follow:
